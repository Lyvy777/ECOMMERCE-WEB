# E-Commerce System for a Supermarket

## Description
This project is a simple e-commerce website for a supermarket.  
It allows users to view products and simulate online shopping.  
The system was developed as a software development project.

## Technologies Used
- Python (backend)
- HTML (structure of web pages)
- CSS (styling and layout)

## Features
- Login and sign up 
- Display supermarket products
- Basic web pages for shopping interface
- Simple backend structure using Python
- Shopping cart and checkout
- Billing address

## How to Run the Project
1. Install Python on your computer.
2. Download or clone the project files.
3. Open the project folder.
4. Run the Python file app.py
5. Open the browser using the local server link (if applicable).

## Project Purpose
This project was created for learning and practice in web development and software development concepts.
